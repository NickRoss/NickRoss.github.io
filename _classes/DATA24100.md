This course equips data science students with practical skills in software engineering, focusing on building complete data pipelines from raw data through relational databases to API deployment. Students work in groups on hands-on projects while learning industry best practices for code quality, testing, and professional development workflows.

Course materials: [2024](https://github.com/NickRoss/2024-Data-24100), [2025](https://github.com/NickRoss/2025-Data-24100).
